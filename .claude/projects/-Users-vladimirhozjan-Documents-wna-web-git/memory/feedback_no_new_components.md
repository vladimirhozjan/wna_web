---
name: No new components without asking
description: User wants to reuse existing components, text classes, and color styles — ask before creating new ones
type: feedback
---

Do not create new components, CSS classes, or color variables without asking first. Reuse existing ones from the codebase.

**Why:** User prefers consistency and minimal file creation. New components add complexity.

**How to apply:** Before adding any new .vue component or CSS variable, check if an existing one can be used. If truly needed, ask the user first.
