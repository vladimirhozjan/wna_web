## 2. Service Description

WhatsNextAction is a web-based productivity platform implementing the Getting Things Done (GTD) methodology. The Service provides tools for capturing, clarifying, organizing, reviewing, and engaging with tasks and projects.

### Core Features

The Service includes the following functionality:

- **Inbox** - Capture raw items for later processing
- **Clarify Workflow** - Guided process for processing inbox items into actions, projects, reference, or someday/maybe items
- **Next Actions** - List of concrete, actionable steps
- **Today** - Focused daily action list
- **Projects** - Multi-step outcomes with ordered action lists
- **Calendar** - Time-specific actions and recurring events (day, week, month, and year views)
- **Waiting For** - Delegated or pending items
- **Someday / Maybe** - Items for future consideration
- **Reference** - File storage for non-actionable materials
- **Weekly Review** - Guided checklist for maintaining your system
- **Recurring Actions** - Automated repeating action templates
- **Context Tags** - Tag-based filtering for context-specific work
- **Quick Add** - Rapid inbox capture from any page

Feature availability depends on your subscription plan (see Section 5).
